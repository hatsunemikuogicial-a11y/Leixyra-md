## 🧠 ¿Qué es LEIXYRA-MD?

💜 **LEIXYRA-MD** es un bot de WhatsApp moderno, rápido y personalizable, diseñado para funcionar 24/7 desde Termux o servidores.

- 🚀 Compatible con Baileys
- 💬 Comandos divertidos y útiles
- 🎮 Sistema RPG y niveles
- 📦 Descargas (videos, música, stickers)
- 🤖 Funciones automáticas
- 🛡️ Protección para grupos (antilink, antispam)
- ⚙️ Totalmente editable a tu gusto
